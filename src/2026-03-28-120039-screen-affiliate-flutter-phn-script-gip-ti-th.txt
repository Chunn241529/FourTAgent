     Context

     User wants to add subtitle handling to Smart Reup Douyin:
     1. Blur existing hardcoded subtitles - user selects region on a video

     FFmpeg command:
     ffmpeg -i input.mp4 -ss 1.0 -vframes 1 -q:v 2 frame.jpg

     ---
     Phase 2: Backend - Blur Subtitle Region

     New transform type: blur_region

     Parameters:
     - blur_x, blur_y - top-left corner of blur region
     - blur_w, blur_h - width/height of blur region

     FFmpeg approach - use delogo or drawbox with heavy blur:

     # Option 1: boxblur with delogo-like effect
     vf = "split=2[bg][sub];[bg]drawbox=x={x}:y={y}:w={w}:h={h}:color=black@1:t=
     fill[bg];[sub]subtitles=subs.srt[sub];[bg][sub]overlay=0:0"

     # Option 2: Simple blur region only (no subtitle burn)
     vf = "boxblur=enable='between(t,0,9999)':x={x}:y={y}:w={w}:h={h}:h=20:v=20"

     Actually simpler - just apply heavy boxblur to the region:

     # Blur a rectangular region
     ffmpeg -i input.mp4 -vf "split=2[bg][vid];[vid]crop={w}:{h}:{x}:{y},boxblur
     =20[blurred];[bg][blurred]overlay={x}:{y}" output.mp4

     ---
     Phase 3: Backend - Burn Subtitles

     New transform type: burn_subtitles

     Parameters:
     - subtitle_file - path to SRT file (already uploaded) OR
     - subtitle_text - plain text to generate timing for
     - subtitle_duration - total duration in seconds (for text mode)

     FFmpeg command for SRT:
     ffmpeg -i input.mp4 -vf
     "subtitles=subs.srt:force_style='FontSize=18,PrimaryColour=&HFFFFFF&'"
     output.mp4

     For text auto-timing:
     - Split text into chunks (by sentences or words)
     - Generate SRT with equal-duration segments
     - Default: 3-5 seconds per chunk based on total duration
